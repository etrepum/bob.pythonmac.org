--- categories: python, simplejson date: 2007/01/18 00:23:53 guid: http://bob.pythonmac.org/?p=213 permalink: http://bob.pythonmac.org/archives/2007/01/18/simplejson-15/ tags: '' title: simplejson 1.5 --- .. -*- mode: rst -*- `simplejson`_ is a simple, fast, complete, correct and extensible `JSON`_ encoder/decoder for Python 2.3+. It is pure Python code with no dependencies. `simplejson`_ 1.5 is a major update that provides better Python 2.5 and Windows compatibility, and two new features that control encoding (``indent`` for pretty-printing, and ``separators`` for generating optimally compact JSON).
