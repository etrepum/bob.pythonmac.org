For those of you interested in learning more about `Erlang`_, I highly recommend buying `Joe Armstrong`_'s `Programming Erlang`_ and/or read his 2003 ph.d thesis, which is kinda similar in content to his book but is more technical and is freely available: `Making Reliable Distributed Systems in the Presence of Software Errors`_. The thesis is a bit more dense, but it was what really made `Erlang`_ "click" for me. If you want to play with the examples, go to `CEAN`_ and download "Erlang/OTP Full (Standard)" at the bottom for you version of Mac OS X and follow the "How do I install CEAN to /usr/local/bin" `FAQ entry`_. Alternatively, if you want to wait longer, you could use `MacPorts`_ to download and compile it. Once you've got that set up just run ``erl`` from the examples directory in the slides download and it *should* all work. .. _`MacPorts`: http://www.macports.org/ .. _`FAQ entry`: http://cean.process-one.net/faq/ .. _`Programming Erlang`: http://www.pragmaticprogrammer.com/titles/jaerlang/ .. _`caturday`: http://en.wikipedia.org/wiki/Lolcat .. _`CEAN`: http://cean.process-one.net/download/ .. _`Erlang`: http://www.erlang.org/ .. _`C4[1]`: http://c4.rentzsch.com/1/ .. _`Joe Armstrong`: http://armstrongonsoftware.blogspot.com/ .. _`Making Reliable Distributed Systems in the Presence of Software Errors`: http://www.sics.se/~joe/thesis/armstrong_thesis_2003.pdf
