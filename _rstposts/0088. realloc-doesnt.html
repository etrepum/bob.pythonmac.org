--- categories: python date: 2005/01/01 22:05:57 guid: /?p=88 permalink: http://bob.pythonmac.org/archives/2005/01/01/realloc-doesnt/ tags: '' title: realloc.. doesn't? --- .. -*- mode: rst -*- Apparently, `Darwin's implementation`_ of ``realloc`` never frees memory if you ask for a smaller size. `Python`_, and probably plenty of other software, assumes that it does, because that's how it behaves on the other platforms (at least Linux and OpenBSD). Oops! .. _`Darwin's implementation`: http://www.opensource.apple.com/darwinsource/10.3.7/Libc-320.1.3/gen/malloc.c .. _`Python`: http://python.org/sf/1092502
