--- categories: python, simplejson date: 2007/03/03 21:40:17 guid: http://bob.pythonmac.org/archives/2007/03/03/simplejson-16/ permalink: http://bob.pythonmac.org/archives/2007/03/03/simplejson-16/ tags: '' title: simplejson 1.6 --- .. -*- mode: rst -*- `simplejson`_ is a simple, fast, complete, correct and extensible `JSON`_ encoder/decoder for Python 2.3+. It is pure Python code with no dependencies. `simplejson`_ 1.6 is a minor update that improves ``str`` support for encoding. Previous versions of simplejson integrated strings directly into the output stream, this version ensures they're of a particular encoding (default is UTF-8) so that the output stream is valid. See the `documentation`_ for more information. .. _`documentation`: http://svn.red-bean.com/bob/simplejson/trunk/docs/index.html .. _`JSON`: http://json.org/ .. _`JSON.org`: http://json.org/ .. _`simplejson`: http://undefined.org/python/#simplejson
