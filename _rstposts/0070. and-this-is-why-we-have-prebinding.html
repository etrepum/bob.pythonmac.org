--- categories: python, py2app date: 2004/04/04 02:43:44 guid: /?p=70 permalink: http://bob.pythonmac.org/archives/2004/04/04/and-this-is-why-we-have-prebinding/ tags: '' title: And this is why we have prebinding... --- .. -*- mode: rst -*- .. original id: P55 I thought it would be interesting to see some `Mach-O`_ dependency graphs, so I whipped up a quick Python script to generate `GraphViz`_ output using my `macholib`_ module. These graphs are built using only the LC_LOAD_DYLIB command. If I were to include the LC_PREBOUND_DYLIB command, they get a whole order of magnitude uglier.
